# Full-Stack Project

## Structure

- `frontend/` - React application with Vite
- `backend/` - Express.js API server

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

### Docker (Optional)
```bash
docker-compose up
```

## Development

- Frontend runs on http://localhost:3000
- Backend runs on http://localhost:5000
- API endpoints available at http://localhost:5000/api
